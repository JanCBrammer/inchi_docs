41/*******************************************************
42 --- Header of tautomers groups ---
43 --- Each entry is AT_TAUTOMER_HDR type ---
44 number of tautomer groups (nNumTautGroups)
45 index of the first tautomer group (#1)
46 ...
47 index of the last tautomer group (#nNumTautGroups)
48 --- end of the Header of tautomers groups description ---
49
50 --- One endpoint group description ---
51 --- Each entry has AT_TAUTOMER type members ---
52 <fixed portion (6 entries)>
53 number of endpoints (0=end of list)
54 number of mobile groups, including number of negative charges (=num(H)+num(-))
55 number of negative charges
56 number of 1H atoms
57 number of 2H (deuterium) atoms
58 number of 3H (tritium) atoms
59 <variable portion, sorted in ascending order>
60 atom rank #1 (ascending order)
61 ...
62 atom rank #endpoints
63 --- end of the endpoint group description ----
64
65 ----------------------------------------------
66 Note:
67 In the following Linear CT Tautomer descriptions
68 we assume the tautomeric groups and the endpoints
69 within them have been properly sorted
70
71 --------- Linear CT Tautomer description -----
72 <for each tautomeric group>
73 -- fixed length part, non-isotopic --
74 number of endpoints = t_group->nNumEndpoints
75 number of mobile atoms = t_group->num[0]
76 ...
77 number of negative charges = t_group->num[T_NUM_NO_ISOTOPIC-1]
78 -- fixed length part, isotopic --
79 number of T (3H) = t_group->num[T_NUM_NO_ISOTOPIC]
80 ...
81 number of 1H = t_group->num[T_NUM_NO_ISOTOPIC+T_NUM_ISOTOPIC-1]
82 -- variable length part --
83 rank of the first endpoint = nRank[t_group_info->nEndpointAtomNumber[t_group->nFirstEndpointAtNoPos]];
84 ...
85 rank of the last endpoint = nRank[t_group_info->nEndpointAtomNumber[t_group->nFirstEndpointAtNoPos+t_group->nNumEndpoints-1]];
86
87 --------- Linear CT Isotopic Tautomer description -----
88 <for each isotopic tautomeric group>
89 number of T (3H) = t_group->num[T_NUM_NO_ISOTOPIC]
90 ...
91 number of 1H = t_group->num[T_NUM_NO_ISOTOPIC+T_NUM_ISOTOPIC-1]
92 t-group ordering number in the Linear CT Tautomer, starts from 1
93
94***************************************************************/
